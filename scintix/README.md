# Scintix — صفحة عرض الشعار

ملفّات المشروع بسيطة — افتح `index.html` في المتصفّح لعرض الشعار.

خطوات سريعة:

1. انسخ الصورة المرفقة من المحادثة وألصقها في المجلد `assets` باسم `logo.png`.
2. افتح الملف `index.html` في المتصفّح (نقر مزدوج أو سحب إلى نافذة المتصفح).

يمكنك تعديل `css/styles.css` لتغيير الألوان أو تخطيط الصفحة.
